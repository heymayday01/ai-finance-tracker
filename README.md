Track income. Audit expenses. Let AI tell you where your money is actually going.

## в—Ҳ API Reference

<br/>

### Authentication

```
POST  /auth/register    вҶ’  Create account         body: { name, email, password }
POST  /auth/login       вҶ’  Get JWT token           body: { email, password }
```

### Transactions  `рҹ”’ Bearer token required`

```
POST    /transactions/          вҶ’  Add transaction      body: { type, amount, category, description }
GET     /transactions/          вҶ’  List all             returns: Transaction[]
GET     /transactions/summary   вҶ’  Totals               returns: { total_income, total_expense, balance }
DELETE  /transactions/{id}      вҶ’  Delete by ID
```

### AI  `рҹ”’ Bearer token required`

```
GET  /ai/insights    вҶ’  Gemini-generated markdown analysis of spending patterns
```

## в—Ҳ Getting Started

<br/>

### Prerequisites

```bash
python --version   # 3.10 or higher
node --version     # 18 or higher
psql --version     # PostgreSQL 14+
```

<br/>

### 1 В· Clone

```bash
git clone https://github.com/heymayday01/ai-finance-tracker.git
cd ai-finance-tracker
```

### 2 В· Backend

```bash
cd backend

# Create and activate virtual environment
python -m venv venv
venv\Scripts\activate          # Windows
source venv/bin/activate       # Mac / Linux

# Install dependencies
pip install -r requirements.txt
```

Create `backend/.env`:

```env
DATABASE_URL=postgresql://postgres:yourpassword@localhost:5432/finance_tracker
SECRET_KEY=your_super_secret_key_here_make_it_long
GEMINI_API_KEY=your_gemini_api_key_here
```

Create the database:

```sql
CREATE DATABASE finance_tracker;
```

Start the server:

```bash
uvicorn app.main:app --reload
# вҶ’ Running at  http://localhost:8000
# вҶ’ Swagger UI  http://localhost:8000/docs
```

### 3 В· Frontend

```bash
cd frontend
npm install
npm run dev
# вҶ’ Running at http://localhost:5173
```
